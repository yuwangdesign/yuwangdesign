CASE STUDIES

SUPREME
WEBSITE REDESIGN

USABILITY TESTING
Enhance user experience, maintain brand personality.

The purpose of this study is to run an expert evaluation on Supremenewyork.com in relation to the following: Heuristics, Accessibility, Performance, Satisfaction, Biometrics, and Psychology. The test result will inform a redesign that aims to be more usable and accessible than the original site. The redesigned site will then be evaluated again, through A/B testing, to see if the Performance and Satisfaction are improved.

Key Findings

Usability Issues

◼ Lack of functionality (search, weak filter, return, favourite goods collection, reviews...)
◼ It doesn't accord with users' common sense of online shopping (image gallery, separate size and delivery info...)
◼ Efficiency issues (changing product attributes in the cart, can't remember the location of the last product that users view...)
◼ Weak navigation
◼ Lack of exciting degree as a shopping website
◼ Lack of standard layouts and effects‍

Accessibility Issues

◼ Text issues (small word size, weak heading and text hierarchy, lack of page title...)
◼ Lack of keyboard accessibility
◼ Colour contrast issues
◼ Lack of responsive design‍

Discussion & Reflection

According to the redesign test outcomes, the redesign was successful (more satisfying, efficient, and effective)
Limitations

◼ The sample size was too small to make a strong parametric statistic test of performance and satisfaction. (Need 2 groups of 26 people at least). At the same time, there were many invalid samples, especially in lostness tests which made the tests even less reliable.
‍
◼ These evaluation tools or principles mainly set up a framework, but whether the analysis is comprehensive requires the joint efforts of multiple members, I didn't arrange a unified time to do the evaluations together.
‍
◼ Prototype design mainly aims to solve the discovered problems and contains insufficient information and few interactive elements, which greatly reduce the task time.
